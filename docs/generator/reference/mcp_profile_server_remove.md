# docker mcp profile server remove

<!---MARKER_GEN_START-->
Remove MCP servers from a profile by server name.

### Aliases

`docker mcp profile server remove`, `docker mcp profile server rm`

### Options

| Name     | Type          | Default | Description                                             |
|:---------|:--------------|:--------|:--------------------------------------------------------|
| `--name` | `stringArray` |         | Server name to remove (can be specified multiple times) |
