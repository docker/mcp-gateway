# docker mcp catalog server remove

<!---MARKER_GEN_START-->
Remove MCP servers from a catalog by server name.

### Aliases

`docker mcp catalog server remove`, `docker mcp catalog server rm`

### Options

| Name     | Type          | Default | Description                                             |
|:---------|:--------------|:--------|:--------------------------------------------------------|
| `--name` | `stringArray` |         | Server name to remove (can be specified multiple times) |
